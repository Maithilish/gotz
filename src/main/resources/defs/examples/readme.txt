this folder contains examples of gotz definition files

folder jsoup contains examples which use JSoup and selectors to query data
folder htmlunit contains examples which uses HtmlUnit and xpath to query data
folder page contains HTML pages used by examples

to run example edit gotz.properties file and add beanFile property with
example bean.xml file to run. To run, jsoup ex-1 add following line :

gotz.beanFile=/defs/examples/jsoup/ex-1/bean.xml
