this folder contains example gotz definition files

jsoup folder - examples which use JSoup and selectors to query data
htmlunit folder - examples which use HtmlUnit and xpath to query data
page folder - HTML pages used by examples

to run examples, edit gotz.properties and add beanFile property with
the example bean.xml file to run. 

To run, jsoup ex-1 add following line

gotz.beanFile=/defs/examples/jsoup/ex-1/bean.xml

Step-by-step guide http://www.codetab.org/gotz-etl explains the definitions
used by the examples.
